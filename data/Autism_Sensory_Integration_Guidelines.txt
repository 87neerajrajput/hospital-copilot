AUTISM & SENSORY INTEGRATION THERAPY GUIDELINES

RECOMMENDED GOALS

Communication & Social Interaction
- Improve joint attention
- Improve eye contact and engagement
- Improve receptive language skills
- Improve expressive communication skills
- Improve social interaction and peer engagement
- Improve turn-taking and sharing skills
- Improve play skills (functional and pretend play)
- Improve imitation skills
- Improve functional communication

Attention & Learning Readiness
- Improve attention span
- Improve sitting tolerance
- Improve task completion
- Improve classroom readiness skills
- Improve following instructions
- Improve cognitive flexibility

Sensory Processing & Regulation
- Improve sensory modulation
- Improve sensory discrimination
- Improve self-regulation
- Improve sensory tolerance
- Improve body awareness
- Improve emotional regulation
- Improve transition tolerance

Motor Planning & Functional Skills
- Improve motor planning (Praxis)
- Improve bilateral coordination
- Improve balance and coordination
- Improve postural control
- Improve adaptive responses
- Improve self-help and daily living skills

Behavioral & Adaptive Skills
- Reduce maladaptive behaviors
- Improve compliance and cooperation
- Improve safety awareness
- Improve independence in daily routines
- Improve executive functioning skills

WEEKLY THERAPY SCHEDULE

Monday: Joint Attention + Sensory Regulation
Tuesday: Communication + Motor Planning
Wednesday: Sensory Processing + Social Skills
Thursday: Attention + Adaptive Skills
Friday: Regulation + Generalization

HOME PROGRAM

Daily (15-20 Minutes)
- Encourage requesting using words, signs, or pictures
- Read one picture book daily
- Practice turn-taking during play
- Animal walks and heavy work activities
- Movement breaks every 1-2 hours
- Puzzle, matching, and sorting activities
- Dressing, feeding, and toy clean-up routines
